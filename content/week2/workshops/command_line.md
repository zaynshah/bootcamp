## Learning Objectives

By the end of this session, an attendee should be able to

- Articulate the Unix philosophy
- Use common built-in Unix command line tools like `touch`, `echo`, `cat`, `head`, `tail` etc.
- Navigate around their local filesystem from the command line
- Find help documentation for arbitrary commands
- Use pipes to combine simple commands for complex results
- Use `grep` or `rg` to find useful information from the output of other commands
- Define aliases within their shell profile file
